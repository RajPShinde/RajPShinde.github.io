Projects

Quadruped Dog Robot

Currently building a 12-DOF Quadruped Robot along the principles of Spot Mini.

  • Modeled the Forward & Inverse Kinematics, Jacobian, Dynamics and Tip Path Planning for the Robot.

  • Implemented and simulated an LQR controller for stable trot gait over uneven surfaces, in MATLAB.

  • Designed CAD Model of the Robot in SOLIDWORKS, for Simulating the robot in V-REP and 3D Printing the parts.

  • Used Brushless Motors with encoders for actuating the all the robot joints.

  • Based on ROS and will use LiDAR-based SLAM for navigation.

Eco-bot

Developed a ROS Package to simulate a Turtlebot in Gazebo to demonstrate a trash-collecting robot.

  • Created Map of the environment using LiDAR-based SLAM and RRT Exploration package.

  • Developed an Astar path planner to reach the trash location, in C++ 11/14 following Agile Development Process.

  • Performed Rostest and Unit testing with googletest, and used catkin as Build System.

  • Documented with Doxygen and performed Continuous Integration(CI) using Travis and Coveralls with 94% coverage.
